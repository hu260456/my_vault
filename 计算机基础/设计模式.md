## SOLID 原则

- 开放封闭原则
    - 对扩展开放
    - 对修改关闭
- 李氏替换原则
    - 所有引用父类的地方须能够平等地使用其子类对象
- 依赖倒转原则
    - 高层模块不应该依赖底层模块
    - 二者都应该依赖其抽象
    - 抽象不应该依赖细节
    - 细节应该依赖抽象
    - 总而言之，针对接口编程，而不是针对实现编程
- 接口隔离原则
    - 使用多个专门的接口，而不使用单一的总接口
    - 即客户端不应该依赖那些它不需要的接口
- 单一职责原则
    - 类的职责单一

## 设计模式

### 创建型模式

- 简单工厂模式
    - 只有一个工厂类
    - 这个类能生产所有相关的产品
    - 此设计模式违背了 `单一职责原则`，`开闭原则`
- 工厂方法模式
    - 定义一个抽象工厂
    - 每个具体产品由一个具体工厂类生产
    - 每增加一个具体产品类，就需要增加一个对应的工厂类
- 抽象工厂模式
    - 相比工厂方法模式，每个具体工厂都生产一套产品
    - 难以添加新的(抽象)产品
    - 建造者模式
        - 将一个复杂对象的构建与其表示分离 

        ```Python

                self.level = level
                self.money = money

                ...

            def build_id(self):
                ...
            @abstractmethod

                ...

                self.money = 999999

            def __init__(self): ...
            def build_player(self, builder):
                builder.build_id()
                builder.build_level()
                builder.build_money()
                return builder.player
        
              
        if __name__ == '__main__':

            director = PlayerDirector()
            rich_man = director.build_player(rich_man_builder)
            print(rich_man)
        ```

- 组合模式
    - 用户对单个对象和组合对象的操作具有一致性

    ```Python
    from abc import ABCMeta, abstractmethod
    
    class Graphic(metaclass=ABCMeta):
        @abstractmethod
        def draw(self):
            ...
    
    class Point(Graphic):
        def __init__(self, x, y):
            self.x = x
            self.y = y
        def __str__(self):
            return f'({self.x}, {self.y})'
        def draw(self):
            print(str(self))
        
    class Line(Graphic):
        def __init__(self, p1, p2):
            self.p1 = p1
            self.p2 = p2
        def __str__(self):
            return f'{self.p1} <-> {self.p2}'
        def draw(self):
            print(str(self))
    
    class Pattern(Graphic):
        def __init__(self, iterable):
            self.children = []
            for i in iterable:
                self.add(i)
        def add(self, graphic):
            self.children.append(graphic)
        def draw(self):
            for i in self.children:
                i.draw()
                
    if __name__ == '__main__':
        p0 = Point(1, 2)
        l0 = Line(Point(2, 3), Point(3, 4))
        pic = Pattern([p0, l0])
        p0.draw()
        l0.draw()
        pic.draw()
    ```

- 享元模式
- 代理模式

    ```Python
    from abc import ABCMeta, abstractmethod
    class Subject(metaclass=ABCMeta):
        @abstractmethod
        def getContent(self):
            ...
        @abstractmethod
        def setContent(self, content):
            ...
            
      
    # 原始对象
    class RealSubject(Subject):
        def __init__(self, filename):
            self.filename = filename
            f = open(filename)
            self.content = f.read()
            f.close()
            
        def get_content(self):
            return self.content
          
        def set_content(self, content):
            f = open(self.filename, 'w')
            f.write(content)
            f.close()
            
            
    # 虚代理        
    class VirtualSubject(Subject):
        def __init__(self, filename):
            self.filename = filename
            self.subj = None
            
        def get_content(self):
            if not self.subj:
                self.subj = RealSubject(self, filename)
            return self.subj.get_content()
          
        def set_content(self, content):
            if not self.subj:
                return self.subj.set_content(content)
              
              
    # 保护代理
    class ProtectedSubject(Subject):
        def __init__(self):
            self.subj = RealSubject(filename)
        
        def get_content(self):
            return self.subj.get_content()
          
        def set_content(self):
            raise PermissionError('无写入权限')
    ```

### 行为型模式

- 解释器模式
- 责任链模式
    - 低耦合度
    - scrapy 的 pipeline 就是责任链模式

    ```Python
    from abc import ABCMeta, abcstractmethod
    class Handler(metaclass=ABCMeta):
        @abcstractmethod
        def handle_leave(self, day):
            ...
    
            
    class GeneralManager(Handler):
        def handle_leave(self, day):
            if day <= 10:
                print('总经理准假%d天' % day)
            else:
                print('你还是辞职吧')
                
                
    class DepartmentManager(Handler):
        def __init__(self):
            self.superior = GeneralManager()
            
        def handle_leave(self, day):
            if day <= 5:
                print('部门经理准假%d天' % day)
            else:
                print('部门经理职权不足')
                self.superior.handle_leave(day)
                
    
    class ProjectDirector(Handler):
        def __init__(self):
            self.superior = DepartmentManager()
            
        def handle_leave(self, day):
            if day <= 1:
                print('项目主管准假%d天' % day)
            else:
                print('项目主管职权不足')
                self.superior.handle_leave(day)
    ```

    ```Python
    from abc import ABCMeta, abstractmethod
    
    # 抽象订阅者
    class Observer(metaclass=ABCMeta):
        @abstractmethod
        def update(self, notice):
            ...
    
    
    # 抽象发布者
    class Notice:
        def __init__(self):
            self.observers = []
            
        def attach(self, obs):
            self.observers.append(obs)
            
        def detach(self, obs):
            self.observers.remove(obs)
            
        def notify(self):
            for obs in self.observers:
                obs.update(self)
    
    
    # 具体发布者
    class StaffNotice(Notice):
        def __init__(self, company_info):
            super().__init__()
            self.__company_info = company_info
            
        @property
        def company_info(self):
            return self.__company_info
    
        @company_info.setter
        def company_info(self, info):
            self.__company_info = info
            self.notify()
            
          
    # 具体订阅者
    class Staff(Observer):
        def __init__(self):
            self.company_info = None
        def update(self, notice):
            self.company_info = notice.company_info
    ```

- 访问者模式
- 模板方法模式
    - 各个子类中的公共行为被提取出来并集中到一个公共父类中以避免代码重复

    ```Python
    from abc import ABCMeta, abcstractmethod
    import time
    
    class Window(metaclass=ABCMeta):
        @abstractmethod
        def start(self):
            ...
            
        @abstractmethod
        def repaint(self):
            ...
            
        @abstractmethod
        def close(self):
            ...
            
        def run(self):  # 模板方法
            self.start()
            while True:
                try:
                    self.repaint()
                    time.sleep(1)
                except KeyboardInterrupt:
                    break
            self.close()
            
    class MyWindow(Window):
        def __init__(self, msg):
            self.msg = msg
            
        def start(self):
            print('窗口初始化')
            
        def repaint(self):
            print(self.msg)
            
        def close(self):
            print('窗口关闭')
            
    # Client
    MyWindow('new window').run()
    ```
